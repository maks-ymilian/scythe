# Basics

## Scythe File Structure
A Scythe `.scy` file is composed of the following (all of which are optional):
- [Sections](language_constructs.md/#sections)
- Global Declarations
- [Input And Description Statements](language_constructs.md/#input-and-description-statements)
- [Import Statements](#import-statements)

### Global Declarations
[Variables](language_constructs.md/#variables) - allowed in code blocks

[Functions](language_constructs.md/#functions) - allowed in code blocks

[Structs](language_constructs.md/#struct-definition) - only allowed at global scope

these will be put in an `@init` section in the compiled output

## Literals
- number literal

- block expressions>

## Operators
- binary including precedence
- unary
